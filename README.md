# Mini-EDA-Titanic-Dataset
This project is about doing EDA on Titanic dataset with more added features which will helped in understanding the underlying facts and vizualization using heatmaps helps in gainig more information about the dataset.

## 🎯 Objectives

- Clean the dataset by handling missing values.
- Remove unnecessary columns.
- Create new features for better analysis.
- Analyze survival rates based on different factors.
- Visualize insights using graphs.

---

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Dataset

**Dataset:** Titanic Dataset

The dataset contains passenger information such as:
- Passenger Class
- Name
- Sex
- Age
- Fare
- Embarked Port
- Family Information
- Survival Status

---

## 🧹 Data Cleaning

The following preprocessing steps were performed:

- Filled missing values in the **Age** column using the mean age.
- Removed the **Cabin** column due to a large number of missing values.
- Created a new **Age Group** column.
- Created a new **Family Size** column using:
  - SibSp (Siblings/Spouses)
  - Parch (Parents/Children)

---

## 📊 Exploratory Data Analysis

The following analyses were performed:

- Survival Rate by Age Group
- Survival Rate by Embarkation Port
- Survival Rate by Family Size

---

## 📈 Visualizations

The project includes:

- Age Distribution Histogram
- Correlation Heatmap
- Survival Rate by Family Size Bar Plot

---

## 📌 Key Insights

- Age distribution shows that most passengers were young adults.
- Survival rates vary across different age groups.
- Family size has an impact on passenger survival.
- Embarkation port also influences survival trends.
- Correlation analysis helps identify relationships between numerical features.
